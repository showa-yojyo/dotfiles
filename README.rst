Installation
============

As this is my private project, I don't want you to have
this git repository cloned to your machine...

.. code:: bash

   $ cd ~
   $ git clone git://github.com/showa-yojyo/bin.git

And make ``$PATH`` enviroment value include ``~/bin``.
